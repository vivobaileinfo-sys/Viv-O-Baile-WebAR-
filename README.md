# Viv’O Baile (WebAR)

Experiência de realidade aumentada interativa para promover a banda Viv’O Baile através de um logótipo físico que ativa conteúdos digitais em AR.

---

## 📌 Descrição

Este projeto permite ao utilizador apontar a câmara do telemóvel para o logótipo da banda presente numa lona de palco e visualizar uma experiência em realidade aumentada com elementos visuais e multimédia.

---

## 🎯 Funcionalidades

- Reconhecimento de imagem (logótipo da banda)
- Modelos 3D interativos:
  - Microfone
  - Guitarra elétrica
  - Teclado
  - Bateria
- Vídeo flutuante em AR
- Interface simples e visual

## 🛠️ Tecnologias Utilizadas

- MindAR (WebAR)

- HTML5 / JavaScript

---

## 🚀 Como usar

1. Abrir o link do projeto no telemóvel  
2. Permitir acesso à câmara  
3. Apontar para o logótipo da banda  
4. Interagir com os elementos em realidade aumentada
